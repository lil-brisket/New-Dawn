# @dawn/config

Shared ESLint, Prettier, and TypeScript configuration for the Dawn monorepo.

## Exports

- `@dawn/config/eslint/base` — base ESLint flat config
- `@dawn/config/eslint/react-native` — React Native app rules
- `@dawn/config/eslint/node` — Node / game-core rules
- `@dawn/config/prettier` — Prettier options
- `@dawn/config/typescript/*` — tsconfig bases
